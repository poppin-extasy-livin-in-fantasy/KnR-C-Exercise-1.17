# KnR-C-Exercise-1.19

Solution to exercise 1.19 from "The C Programming Language" by Kernighan & Ritchie.
Reverses each input line using a custom `reverse()` function.
Written in C (C11), compiled with MSVC (Visual Studio).

## Example

Input:

hello, world


Output:

dlrow ,olleh
